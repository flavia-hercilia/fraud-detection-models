# Fraud Detection ML Comparison

Este repositório contém um estudo comparativo de modelos de Machine Learning aplicados à identificação e classificação de fraudes em um conjunto de dados (dataset). O objetivo principal é analisar o desempenho de diferentes algoritmos para lidar com o problema clássico de detecção de fraudes, que geralmente envolve dados altamente desbalanceados.
